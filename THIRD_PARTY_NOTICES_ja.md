# 第三者ライセンス情報

Viruneは以下の第三者npmパッケージを利用します。これらは独立したnpm依存としてインストールされ、Viruneが生成するプログラムへソースコードを埋め込みません。

| パッケージ | バージョン | ライセンス | 用途 |
|---|---:|---|---|
| TypeScript | 6.0.3 | Apache-2.0 | TypeScript宣言解析と開発ツール |
| Chevrotainおよび関連scoped package | 12.0.0 | Apache-2.0 | Lexer／Parser実装 |
| `@jridgewell/gen-mapping` | 0.3.13 | MIT | Source Map生成 |
| `@jridgewell/trace-mapping` | 0.3.31 | MIT | Source Map検証 |
| `@jridgewell/resolve-uri` | 3.1.2 | MIT | Source Map関連の推移依存 |
| `@jridgewell/sourcemap-codec` | 1.5.5 | MIT | Source Map関連の推移依存 |
| `@types/node` | 24.13.3 | MIT | 開発時の型定義 |
| `undici-types` | 7.18.2 | MIT | 開発時の推移型定義 |

各ライセンス全文はインストールされたnpmパッケージ内で確認できます。Virune自体はMIT Licenseです。
